# 🖥️ HyperV Backup Manager

Десктопное приложение для автоматического резервного копирования виртуальных машин Hyper-V. Написано на Python + PyQt6, работает только на Windows с установленной ролью Hyper-V.

## Зачем это нужно

Стандартные средства Hyper-V не предоставляют удобного GUI для настройки расписания бэкапов с уведомлениями. Этот инструмент закрывает задачу полностью: выбираешь VM, задаёшь расписание, указываешь куда складывать — дальше всё работает само.

---

## Возможности

### Управление виртуальными машинами
- Отображение всех VM на хосте: имя, статус, память, CPU
- Статусы в реальном времени: Running / Off / Paused / Saved / Starting / Stopping
- Ручной запуск бэкапа для выбранных VM в один клик

### Расписание
- Ежедневные, еженедельные, ежемесячные задания
- Настройка точного времени запуска (часы:минуты)
- Два режима: все VM в одну папку или каждая VM в отдельную подпапку
- Включение / отключение заданий без их удаления

### Email-уведомления
- Отправка отчёта после каждого бэкапа
- Настраиваемые триггеры: успех / ошибка / предупреждение / удаление старых бэкапов
- Поддержка SMTP с STARTTLS (порт 587) и SSL/TLS (порт 465)
- Опциональное вложение лог-файла к письму
- Тестовая отправка прямо из настроек

### История и логи
- Журнал всех операций с цветовой маркировкой по уровням (SUCCESS / ERROR / WARNING / INFO)
- Просмотр истории бэкапов: дата, VM, статус, размер, путь
- Подсчёт размера папок с бэкапами в фоновом потоке

### Прочее
- Тёмная тема (тёмно-серая, indigo-акценты)
- Сворачивание в системный трей — работает в фоне
- Автозапуск при входе в Windows (через реестр)
- Конфигурация хранится в JSON-файлах рядом с приложением

## Установка и запуск

```bash
# 1. Клонировать репозиторий
git clone https://github.com/lukomsky85/hyperv_backup.git
cd hyperv_backup

# 2. Установить зависимости
pip install PyQt6 qtawesome schedule

# 3. Запустить (от имени администратора)
python hyperv_backup.py
```

> ⚠️ Запускать от имени администратора — иначе Hyper-V API недоступен.

---

## Структура конфигурации

Приложение создаёт три файла в папке запуска:

```
hyperv_backup_config.json   — основные настройки (пути, email, параметры)
hyperv_backup_schedule.json — расписание заданий
last_backup.json            — данные о последнем выполненном бэкапе
```

## Стек

- **PyQt6** — GUI, системный трей, потоки
- **qtawesome** — иконки (Font Awesome 5)
- **schedule** — планировщик заданий
- **subprocess + PowerShell** — взаимодействие с Hyper-V API (`Get-VM`, `Export-VM`)
- **smtplib** — отправка email-уведомлений
